== Description ==

RankingBadge displays ranking information from major sources such as Google (PageRank), Alexa (Alexa traffic Rank) and Technorati in the sidebar of your blog.

Check out more [Wordpress Plugins](http://www.grobekelle.de/wordpress-plugins "Wordpress Plugins") brought to you by [Grobekelle](http://www.grobekelle.de "Grobekelle").

== Installation ==

1. Unpack the zipfile rankingbadge-X.y.zip
1. Upload folder `rankingbadge` to the `/wp-content/plugins/` directory
1. Make cache directory writeable chmod 777
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to settings -> RankingBadge and configure colors etc.
1. Place `<?php if(function_exists('rankingbadge_display'))rankingbadge_display(); ?>` in your template or use the sidebar widgets.
